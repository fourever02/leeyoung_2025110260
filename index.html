책 소개

이 책은 웹 개발의 가장 기본이 되는 HTML5를 기초부터 실전 응용까지 누구나 쉽게 배울 수 있도록 구성되었습니다. 비전공자도 이해할 수 있는 쉬운 설명과 풍부한 예제 코드를 통해 나만의 웹사이트를 직접 만들어보는 즐거움을 선사합니다.

주요 특징

  • 최신 HTML5 표준 및 시맨틱 태그 완벽 반영
  • 단계별 실습 예제로 배우는 웹 구조 설계
  • 초보자가 자주 실수하는 포인트 짚어주기
  • 모바일 환경을 고려한 기초 레이아웃 가이드

목차

  1. 웹의 역사와 HTML의 이해
  2. 텍스트와 하이퍼링크 태그
  3. 멀티미디어(이미지, 영상) 활용하기
  4. 데이터 정리를 위한 표와 리스트
  5. 사용자 입력을 받는 폼(Form) 요소
  6. 시맨틱 태그로 완성하는 웹사이트 구조
